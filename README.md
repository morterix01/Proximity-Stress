# StressWatch - Wear OS Premium Watch Face

StressWatch è un'applicazione avanzata per Wear OS che trasforma il tuo smartwatch in un sistema intelligente di monitoraggio e gestione dello stress in tempo reale.

## 🚀 Funzionalità Premium

### ⌚ Watch Face Dinamica
* **Colori Bio-Feedback**: L'orario cambia colore (Cyan, Green, Yellow, Red) in base al tuo livello di stress attuale.
* **Effetto Glow Neon**: Un design moderno e futuristico con bagliore adattivo.
* **Arco dello Stress**: Un indicatore circolare pulsante che mostra visivamente il tuo score di stress (0-100).
* **4 Slot Complicazioni**: Supporto completo per aggiungere meteo, contapassi, batteria o altre app di sistema.
* **Interattività**: Tocca il centro del quadrante per aprire istantaneamente l'app completa.
* **Personalizzazione**: Scegli tra la modalità dinamica o un look bianco classico tramite le impostazioni dell'orologio.

### 🧠 Monitoraggio Intelligente
* **Algoritmo Clinico**: Utilizza il calcolo RMSSD (Heart Rate Variability) per una precisione medica.
* **Alert Aptici**: Vibrazione discreta sul polso quando viene rilevato un picco di stress elevato.
* **Background Monitoring**: Monitoraggio continuo garantito da un Foreground Service, anche a schermo spento.

### 🧘 Gestione dello Stress
* **Esercizio di Respirazione**: Un'interfaccia guidata con animazioni per aiutarti a ritrovare la calma durante i momenti di panico.
* **Analisi Storica**: Grafici dettagliati dell'andamento dello stress nelle ultime ore.
* **Radar in Tempo Reale**: Visualizzazione grafica dinamica dei parametri biometrici.

## 🛠️ Requisiti Tecnici
* **Dispositivo**: Wear OS 3.0 o superiore (es. Galaxy Watch 4/5/6, Pixel Watch).
* **Sensori**: Sensore di frequenza cardiaca (PPG).
* **Permessi**: Body Sensors, Activity Recognition.

## 📦 Installazione
1. Clona la repository.
2. Apri il progetto in Android Studio.
3. Esegui `./gradlew assembleDebug` per generare l'APK.
4. Installa tramite ADB o direttamente da Android Studio sul tuo dispositivo Wear OS.
